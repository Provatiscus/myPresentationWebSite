Style Transfer


Style Transfer is the process of extracting the style of a picture to apply it on another picture.
To do that, we use a Deep Neural Network trained for classification of images. This Network picks up some patterns in the image in order to classify the images. We can recapture these kind of patterns in the deep layers of the Network and we can average them by taking the gram matrix of these layers. This gives us something representing the "style" of the image. Once we have extracted these features we can take the same Network and freeze all its layers but the first one and apply gradient descent in order for it to modify the original image to make the features closer to those extracted previously.

Generative Adversarial Network


Generative Adversarial Network is a special type of network composed of a generator of images and a discriminator which are training against each other.
The generator is a network producing images and the discriminator is a network evaluating how well the images produced by the generator ressemble the reference images. When the generator produces an image, the discriminator evaluates the image and by backward propagation informs the generator on how its weights should be transformed in order to produce a better image. There are a lot of variations for GANS. This explanation is about the simplest form of gans.


The code I wrote for these GANs Production is available on my Github

An Interesting fact that we can point here is how the convolutional neural network can learn perfectly the black background in comparison to the dense neural network. Indeed the convolution can learn spatial representations by isolating some parts of the image. Thus it can learn that some parts have to be dark. In comparison the Dense Neural network sees all its weights interlieved and can't learn some dark part of the background without changing the light part of the foreground.
